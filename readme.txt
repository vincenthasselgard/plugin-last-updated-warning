This plugin will display a warning for plugins that haven't received updates on the WP.org plugin repo the past year.

== Description ==

If you find it cumbersome to manually check if installed plugins have received updated from the developers this is the plugin for you. Install and activate and you'll get a simple notification warning you about an old plugin in your installed plugins list.

== Installation ==

1. Upload the folder `plugin-last-updated-warning` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
